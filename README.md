# AI-CORE: NEXUS DE INTELIGENCIA Y PROTOCOLO DE AUDITORIA ARIMAC

Este repositorio constituye el nucleo de control y estandarizacion para el ecosistema MarIA. Su proposito es actuar como el orquestador tecnico superior, encargado de auditar la calidad del codigo, gestionar la infraestructura y sincronizar la base de conocimiento avanzada mediante el protocolo RAG con NotebookLM.

## Capacidades Tecnicas del Agente (Elite Skills)

### Protocolo Brain-Sync (RAG Dinamico)

El sistema opera bajo una arquitectura de memoria externa persistente. Antes de realizar cualquier propuesta estructural, el agente consulta el workspace de NotebookLM para garantizar alineacion con la documentacion historica.

### Project Superpower (Regla 11)

Autonomia total para auditorias preventivas. El agente identifica y corrige cuellos de botella (bloqueos de asyncio, deuda tecnica) en el momento en que se detectan, sin esperar aprobacion para fallos criticos de rendimiento.

### UI/UX Pro Max (Regla 10)

Aplicacion obligatoria de Atomic Design y micro-interacciones. Cada cambio en el Frontend debe elevar el estandar visual y de accesibilidad de forma proactiva.

### Everything Claude Code (Regla 12)

Gestion completa del entorno. El agente actualiza manifiestos de dependencias y configuraciones de entorno (.env.example) automaticamente tras cambios en la logica del sistema.

## Protocolo de Operacion

1. **Auditoria de Equipo:** Analisis de commits y reporte de 'Must-Fix'.
2. **Sincronizacion de Memoria:** Generacion del 'State of the Union' para NotebookLM.
3. **Inicializacion de Proyecto:** Configuracion automatica de memoria para nuevos repositorios.
